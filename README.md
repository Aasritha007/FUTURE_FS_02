# 🚀 Mini CRM – Client Lead Management System

This is a simple web-based Mini CRM (Customer Relationship Management) system built as a beginner project.

It helps businesses store and manage client leads in an easy way.

---

## 🎯 Project Objective

The goal of this project is to understand how real-world businesses:

- Store customer leads  
- Manage client data  
- Track basic information  

---

## ✨ Features

✔ Add client details (Name, Email)  
✔ Display all leads dynamically  
✔ Simple and clean user interface  
✔ Beginner-friendly project structure  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
